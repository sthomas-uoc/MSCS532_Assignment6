# About

This repository contains the source code for implementation of Medians and Order Statistics & Elementary Data 
Structures and report for Assignment 6: Medians and Order Statistics & Elementary Data Structures

# Running the code

The code contains tests for the selection algorithms and the data structures

To run the test runner for testing the selection algorithms, execute ```python tester.py```

To demonstrate the data structures and operations on them, execute ```python datatypes.py```
